# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Desaturated Red: hsl(0, 36%, 70%)
- Soft Red: hsl(0, 93%, 68%)

### Neutral

- Dark Grayish Red: hsl(0, 6%, 24%)

### Gradients

- Linear, 135deg, from hsl(0, 0%, 100%), to hsl(0, 100%, 98%) - for the background
- Linear, 135deg, from hsl(0, 80%, 86%), to hsl(0, 74%, 74%) - for the button

## Typography

### Body Copy

- Font size: 16px

### Font

- Family: [Josefin Sans](https://fonts.google.com/specimen/Josefin+Sans)
- Weights: 300, 400, 600

### Methods:

- Layout: used CSS Grid https://gridbyexample.com/examples/example5/
- Layout each box according using line placement to achieve desired placement and ratio on page
